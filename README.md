# Global Breathing Map

## Introduction
The Global Breathing Map is a web-based real-time interactive installation that converts the user behaviors scattered around the world into visualized collective traces. When visitors click on any location on the map, a highlighted light point with a pulsating effect like breathing will be generated. All online users can observe the distribution and changes of these light points in real time. This work aims to explore the collective presence and distributed connections in the digital age, transforming intangible network interactions into visible geographical marks.
